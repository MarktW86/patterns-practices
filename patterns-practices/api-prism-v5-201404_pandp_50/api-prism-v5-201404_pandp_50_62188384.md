Prism Class Library

MefFileModuleTypeLoader Class
=============================

Loads modules from an arbitrary location on the filesystem. This typeloader is only called if [ModuleInfo](https://msdn.microsoft.com/t:microsoft.practices.prism.modularity.moduleinfo) classes have a Ref parameter that starts with "file://". This class is only used on the Desktop version of the Prism Library when used with Managed Extensibility Framework.
